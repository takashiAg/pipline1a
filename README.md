## Usage
通常時
```bash:bash
git clone https://github.com/takashiAg/pipline1a
cd pipline1a
npm install
npm start
```

デーモン化するときは
```
git clone https://github.com/takashiAg/pipline1a
cd pipline1a
npm install
npm f-start
# 止めるとき
npm f-stop
```
